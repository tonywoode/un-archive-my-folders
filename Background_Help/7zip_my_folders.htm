  2. ikxcsshcm7

    2009-04-09 15:50:27 UTC
    Look at:
    FOR /D %I IN (*) DO 7z a %I.zip -r ".\%I\*"

    Look at:
    FOR /?

    Look at help .CHM file examples given:

    Command Line Version
    |> Commands
    ||>> a (Add)

    Command Line Version
    |> Switches
    ||>> -r (Recurse subdirectories)
